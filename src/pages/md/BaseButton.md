# 按钮组件
```
<BaseButton>Click</BaseButton>
```
## Props
  round
  >?:boolean是否半圆风格

  type
  > ?:'default' | 'text' | 'line' | 'round-line-left' | 'round-line-right' 风格

  bgStyle
  > ?:'primary' | 'secondary' 蓝色 绿色

  disabled
  >?:boolean 禁用

  loading
  >?:boolean 加载中

  size
  >?:'xs' | 'sm' | 'md' | 'lg' | 'xl' 尺寸

  shadow
  > ?:boolean 阴影

## Slot
默认 按钮内容

## Event
无
